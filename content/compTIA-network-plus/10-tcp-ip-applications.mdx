## ðŸ”— TCP and UDP

TCP and UDP are the main **transport layer** protocols in the TCP/IP suite.

### ðŸ§± TCP (Transmission Control Protocol)
- **Connection-oriented**
- **Reliable** â€” guarantees delivery using acknowledgments
- **Ordered** delivery
- Uses **3-way handshake** to establish connection

### ðŸš€ UDP (User Datagram Protocol)
- **Connectionless**

## ðŸš¦ Network Traffic Types & Flow

### ðŸ§± Traffic Units  
- **Bits** â€“ Basic binary data (0s and 1s)  
- **Frames** â€“ Layer 2 unit (Data Link)  
- **Packets** â€“ Layer 3 unit (Network)  
- **Segments/Datagrams** â€“ Layer 4 unit (Transport)  
- **PDU** â€“ Protocol Data Unit, generic data term for each layer

### ðŸ§­ Traffic Types  
| Type        | Description |
|-------------|-------------|
| **Unicast**   | One-to-one communication (most internet traffic) |
| **Broadcast** | One-to-all communication within a local subnet |
| **Multicast** | One-to-many selective delivery (e.g. live streaming) |
| **Anycast**   | One-to-nearest node (e.g. CDN edge server delivery) |
